# Security Policy

## Reporting a Vulnerability

If you find a security issue in this codebase, **please don't open a public
GitHub issue**. Reach out privately first so the issue can be fixed before
it's disclosed.

**Contact:** info@almanyar.com

Please include:
- A short description of the issue and its impact
- Steps to reproduce (a minimal proof-of-concept is ideal)
- Your name / handle if you'd like credit in the fix commit

A response should come within a few business days. Coordinated disclosure
is appreciated — give the maintainer reasonable time to patch before
publishing details.

## Scope

In scope:
- This codebase (Next.js app, Prisma schema, Docker config, deployment scripts)
- The production deployment at almanyar.com

Out of scope:
- Social-engineering attempts against staff or customers
- Denial-of-service attacks
- Issues in third-party dependencies (please report those upstream)
- Findings that require an already-compromised host or admin account

## What to expect

- Acknowledgement of your report
- An estimate of when a fix will land
- A credit in the fix commit / release notes if you'd like one
